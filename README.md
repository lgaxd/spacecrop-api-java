# 🌾 SpaceCrop
## Monitoramento Agrícola via Satélite

### 📌 Sobre o Projeto
SpaceCrop é uma plataforma tecnológica desenvolvida para a Global Solution 2026/1 da FIAP, com o objetivo de democratizar o acesso a tecnologias de agricultura de precisão.

A solução conecta dados de satélites orbitais (Landsat-9, Sentinel-2, CBERS-4 e GOES-16) a pequenos e médios produtores rurais, permitindo o monitoramento de temperatura, umidade, precipitação e índices de vegetação sem a necessidade de instalação de sensores em campo.

### 🚧 Status do Projeto
**Em desenvolvimento**

> O projeto está sendo desenvolvido como parte das disciplinas do curso de Análise e Desenvolvimento de Sistemas da FIAP.

### 🎯 Objetivo
Fornecer alertas automáticos e relatórios gerenciais baseados em dados coletados por satélites, ajudando o produtor rural a tomar decisões mais assertivas e reduzir perdas na lavoura.

### 📊 Funcionalidades Planejadas
- ✅ Cadastro de usuários, fazendas e setores de plantio
- ✅ Monitoramento via sensores orbitais
- ✅ Geração automática de alertas para condições críticas
- ✅ Relatórios gerenciais com análises por cultura e fazenda
- 🔄 Aplicativo mobile para acompanhamento em tempo real (em desenvolvimento)
- 🔄 API REST para integração com frontend (em desenvolvimento)

### 👨‍🌾 Público-alvo
Pequenos e médios produtores rurais que buscam uma solução acessível e eficiente para monitoramento de suas plantações.
